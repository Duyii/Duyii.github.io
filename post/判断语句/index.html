Python--判断语句

Jan 2, 2020
1 min read

1.判断语句

1.1 if判断语句

  • 作用:控制代码是否被Python执行
  • 格式: 
    if 判断条件:
    如果判断条件成立执行的代码

1.2 if-else

  • 格式: 
    if 判断条件:
    如果条件成立执行的代码
else:
    如果条件不成立执行的代码

1.3 if-elif-else

  • 作用:描述多个判断条件成立后的,执行的不同的代码
  • 格式: 
    if 判断条件1:
    成立,执行的代码
elif 判断条件2:
    成立,执行的代码
elif 判断条件3:
    成立,执行的代码
    ...
else:
    如果条件1,2,3...均不成立,执行的代码

2.三目(元)运算符

  • 作用:把多行代码做的事情,通过一行代码解决
  • 格式: 
    变量 = 成立的数值 if 判断条件 else 不成立的数值

3.if嵌套

  • 概念:如果 if 语句中又包含了 if 语句,就称为 if 嵌套
  • 格式: 
    if 判断条件1:
    如果条件1成立,执行的代码
    if 判断条件2:
        如果条件2成立,执行的代码
else:
    如果条件1不成立,执行的代码
    if 判断条件3:
        如果条件3成立,执行的代码

4.循环语句

4.1 while循环

  • 作用:让指定代码重复执行
  • 格式: 
    # 定义一个计数器
i = 0
# 定义一个while循环
while i < 3:
    重复执行的代码
    # 修改计数器
    i += 1
  • 注意:
    • 即使循环结束,计数器的数值依然存在
    • 如果一个循环的判断条件一直成立,会成为死循环

4.2 while循环嵌套

  • 格式: 
    while 条件1:
    条件1成立,执行的代码
    while 条件2:
        条件2成立,执行的代码

4.3 for循环

  • 格式: 
    for 临时变量 in 字符串等可迭代对象&range函数
  • 循环分类
    • while循环
      • 死循环
    • for 循环
      • 循环遍历可迭代对象
        • 可迭代对象:只要可以使用for循环进行循环遍历的对象就是可迭代对象
        • 对象:变量(万物皆对象)
  • range用法
    • range(a)
      • [0,a)
    • range(a,b)
      • [a,b)]
  • random.randint用法
    • randint(a,b)
      • [a,b]

4.4 break和continue

  • 配合循环(while & for)使用
  • 特点:
    • break:终止循环
    • continue:提前结束本次循环,马上进入下次循环

4.5 for-else

  • 格式 
    for 临时变量 in 字符串等可迭代对象&range
    循环成立执行的代码
else:
    当for循环正常结束后,将执行的代码
  • 总结
    • 默认情况下,监听for循环正常结束后,将执行else中的代码
    • 如果循环中执行了break异常的终止循环,那么else中代码将不再执行
